# Détection d'anomalies réseau par Clustering

## Objectif
Analyser des flux réseau et détecter des anomalies en utilisant des méthodes de clustering.

## Méthodes utilisées

- K-Means
- Clustering hiérarchique
- DBSCAN

## Pipeline

1. Chargement des données
2. Normalisation
3. Réduction de dimension (PCA)
4. Clustering
5. Visualisation
6. Détection d'anomalies
